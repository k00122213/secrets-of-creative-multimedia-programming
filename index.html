Creative Multimedia Programming

Welcome to The Secrets of Creative Multimedia Programming

This is a blog all about Creative Multimedia Programming, a Higher Diploma course focused on developing ICT Skills! The blog will include information that can already be found on the internet in one easy to access location, as well as my own input, opinion and some behind the scenes info about the course!

Below are links and descriptions to the different subjects covered over three semesters (the third semester is a work placement program). The order of this list is randomised but if you would like to see a semesterised breakdown of subjects please click here or alternatively the 'semesters' option in the naviagtion bar.

Web Development

Web Development starts with the basics and teaches you everything you need to know websites.

3D Graphics

This module covers the theory and practical aspects of building 3d objects and animation using 3DSMax

Multimedia Databases

This subject covers a lot of theory on databases and tables including learning some Structured Query Language (SQL).

Interactive Design

Interactive Design covers design theory and best practices and introduces the use of the Adobe CC package for design work.

Web Programming

'Hello World!' You will dive right in to the Javascript language in the first semester of Web Programming.

Advanced Web Programming

Advanced Web Programming covers PHP and the combination of a front end website to back end functionality using Code Igniter!

Human Cognition & Comm

Mostly theory but incredibly insightful study.

Multimedia Project

This module is dedicated to a personal multimedia project builtfrom the ground up by the student.

Evolving Web Technologies

This module covers Responsive Web Design, Search Engine Optimization and a higher level of Web Development

Digital Video

Digital Video consists of team work video production projects that allow you to get creative!

Work Placement

This is a moduleset aside for the entire summer in which we take up a placement position in a company.

Interactive Interface Design

This subject builds on knowledge gained in Interface Design in Semester 1 and covers the UI/UX design and development.
